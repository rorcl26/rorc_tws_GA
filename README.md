# 台灣股市自動分析與通知系統 (5MA 策略)

這個專案每天為您下載台灣股市最新報價，並利用 5MA 移動平均線計算作多、放空、投資區間與風險建議，找出股價在 50~300 元間的前 200 檔優質股票，最後將分析成果以 HTML 格式寄到您的信箱。

## 🚀 快速上手

### 1. 準備金鑰 (.env)
請複製目錄下的 `.env.example`，重新命名為 `.env`，並填寫您的 Gmail 與應用程式密碼。
*請注意：您必須在 Google 帳戶設定中開啟 [兩步驗證] 後，才能產生 16 碼的 [應用程式密碼]。*

### 2. 環境安裝與手動測試
直接點兩下執行 `run_analysis.bat`：
- 若您尚未安裝必要套件，它會自動讀取 `requirements.txt` 安裝。
- 安裝完後會自動執行 `stock_analyzer.py` 並嘗試發送 Email。
- 檢查信箱是否收到報表。如未順利寄信，也會在此資料夾生成一份 `report.html` 供您檢視。

### 3. 設定自動排程 (每日 17:30 執行)
若要讓電腦每天 17:30 (收盤後 4 小時) 自動執行：
1. **右鍵點擊** `setup_task.ps1`，選擇 **「以 PowerShell 執行 (系統管理員)」**。
2. 腳本會自動將其加入 Windows 內建的工作排程器中。
3. *備註：程式內建判斷是否為六日的邏輯，若為六日將自動跳過不寄信。*

## 📈 分析邏輯與指標
- **資料來源**: 今日報價與當沖量取自 [台灣證券交易所與櫃買中心]，歷史資料(用於計算均線)取自 [Yahoo Finance]。
- **核心指標**: 5日移動平均線 (5MA)
  - **作多**: 收盤價 > 5MA，且 5MA 呈現上揚趨勢
  - **放空**: 收盤價 < 5MA，且 5MA 呈現下跌趨勢
- **風險評估**: 計算收盤價與 5MA 的乖離率，或最近振幅 ATR 若過大，則標示為高風險。
- **推估投資區間**: 5MA ± 5日平均真實波幅(ATR)。
